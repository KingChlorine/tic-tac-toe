# tic-tac-toe

Adapted from Medium's guide
"How to create a Tic Tac Toe game with HTML, CSS and Javascript" - 2023
Author - Cana Korkut

Added features:
Building on the code from this tutorial, I added through lines over the winning squares for that classic x's and o'x/tic tac toe winning pencil strike.

I also noticed that after winning empty squares can still be filled, so I made it so that the eventListener inside main game loop ended once a win had been reached by switching gameOver variable between true and false.
